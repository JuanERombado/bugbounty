# Knowledge Base

Drop reference material here for the workbench to ingest.

Use `resource-urls.txt` as the curated source list, then place downloaded PDFs, notes, writeups, or copied text into the matching folders.

## Folders

- `audit-reports/`: public audits and review reports.
- `bug-classes/`: reusable vulnerability classes and checklists.
- `incident-writeups/`: exploit postmortems and public incident analyses.
- `tools/`: tool docs, local setup notes, and usage recipes.
- `training/`: CTFs, courses, labs, and practice material.
- `notes-to-ingest/`: raw notes waiting to be summarized into the taxonomy.

## Rule

This knowledge base supports local-only research and responsible reporting; it should not contain exploit automation for live systems.
