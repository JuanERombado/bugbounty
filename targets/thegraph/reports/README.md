# Reports

Only draft reports for confirmed issues with a local reproducible PoC.

Minimum report sections:

- Title
- Summary
- In-scope asset
- Accepted impact
- Affected code
- Root cause
- Local reproduction steps
- PoC code
- Expected vs actual result
- Impact explanation
- Suggested fix
- Scope and rules check

One-sentence rule: no confirmed local PoC means no report draft.
