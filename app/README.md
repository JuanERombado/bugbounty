# GUI Workbench

Open the local GUI with:

```powershell
python -m backend.hotspot_hub.server --host 127.0.0.1 --port 4173
```

Then visit `http://localhost:4173`.

The GUI is a visualization layer for the workflow. It can initialize local target folders from Immunefi program URLs, but it does not attack live systems and does not run scanners.
