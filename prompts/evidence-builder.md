# Evidence Builder Prompt

Build the evidence dossier for a potential Immunefi report.

Rules:

- Do not write a report yet.
- Mark unknowns clearly.
- Reject the finding if the asset, impact, rules, or PoC evidence are not strong enough.
- Treat AI output as notes; the human researcher makes the final call.

Output:

- Asset and scope evidence:
- Accepted impact evidence:
- Out-of-scope and prohibited-activity checks:
- Known-issue and audit checks:
- Root cause:
- Local PoC command:
- Local PoC files:
- Expected result:
- Actual result:
- Reproduction environment:
- Missing evidence:
- Submission readiness: `not ready | maybe ready | ready for draft`

Input:

```text
[paste scope entry, hypothesis, code references, PoC command, and test output]
```
